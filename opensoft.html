  • Source code for IMRSim

    IMRSim is a simulator that accurately models the emerging interlaced magnetic recording (IMR) disks. Please cite the following accompanying article if you would like to use or build upon the tools:

    IMRSim: A Disk Simulator for Interlaced Magnetic Recording Technology
    Zhimin Zeng, Xinyu Chen, Laurence T. Yang, Jinhua Cui.
    The IFIP International Conference on Network and Parallel Computing (NPC'22), Jinan, China, 2022.
